# Kamon Console

Play the game of Kamon on the console

## Requirements

- Node.js

## Installation

```sh
yarn
```

Also, you can run make 
```sh
make install
```

### Playing

```sh
yarn start
```

Also, you can run make 
```sh
make run
```

### Developing

```sh
## Re-run the script on change
yarn dev

## typecheck && compile
yarn build

## format
yarn prettier --write .

## test
yarn test
```
